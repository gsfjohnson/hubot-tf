hubot-tf
========

Manage tf via hubot.

## Installation

### Update the files to include the hubot-tf module:

#### package.json
    ...
    "dependencies": {
      ...
      "hubot-tf": ">= 0.0.1"
      ...
    },
    ...

#### external-scripts.json
    [...,"hubot-tf"]

Run `npm install` to install hubot-tf and dependencies.
